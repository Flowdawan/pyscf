6.21.2. pbc.ao2mo — Integral transformations for crystalline Gaussian type orbitals¶

The pbc.ao2mo module is an analog of the ao2mo module but with the Gaussian type atomic orbitals replaced by the corresponding crystalline orbitals.

6.21.2.1. Examples¶

  • examples/pbc/30-mo_integrals.py

  • examples/pbc/41-pbc1d_real_space_sum.py

6.21.2.2. Program reference¶

This ao2mo module is kept for backward compatiblity. It’s recommended to use pyscf.pbc.df module to get 2e MO integrals
